# 세무회계청천 홈페이지

인천 송도 연수구에 위치한 **세무회계청천 김지수 세무사** 공식 소개 및 상담 접수 페이지입니다.

---

## ✅ 현재 구현된 기능

| 섹션 | 내용 |
|------|------|
| **Header / Nav** | 고정 상단 네비게이션, 스크롤 시 쉐도우 처리, 모바일 햄버거 메뉴 |
| **Hero** | 메인 슬로건 + 상담신청·전화 CTA 버튼, 사무소 기본 정보 카드 |
| **사무소 소개 (About)** | 김지수 세무사 소개, 주소·전화·카카오·블로그 링크 |
| **서비스 안내 (Services)** | 6가지 서비스 카드 – 카드 Fade-in 애니메이션 포함 |
| **이용 안내 (Notice)** | 업무 시간표, 상담 준비 사항, 오시는 길 (네이버지도 링크) |
| **FAQ** | 자주 묻는 질문 6개 (아코디언 방식) |
| **상담 신청 폼 (Consult)** | 이름·연락처·상담유형·희망시간·문의내용 → `consultations` 테이블 저장 |
| **플로팅 버튼** | 전화 / 카카오톡 / 맨위로 (스크롤 시 등장) |
| **Footer** | 연락처·링크 모음, 저작권 표시 |

## 🔗 진입 URI / 앵커

| 앵커 | 설명 |
|------|------|
| `/#hero` | 메인 히어로 섹션 |
| `/#about` | 사무소 소개 |
| `/#services` | 서비스 안내 |
| `/#notice` | 이용 안내 (업무시간·오시는 길) |
| `/#faq` | 자주 묻는 질문 |
| `/#consult` | 상담 신청 폼 |

## 📦 데이터 모델

### `consultations` 테이블

| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | UUID (자동 생성) |
| name | text | 신청자 이름 |
| phone | text | 연락처 |
| consultation_type | text | 상담 유형 |
| preferred_time | text | 희망 상담 시간 |
| message | rich_text | 문의 내용 |
| status | text | 처리 상태 (접수/확인중/완료) |
| created_at | datetime | 접수 일시 |
